# Crypto-Convertor
## Crypto Converter is a web application that allows users to easily convert between different cryptocurrencies and fiat currencies. Users can enter a value in one currency, and the application will display the equivalent value in other currencies. <br> 

UI repo 
[here](https://github.com/Mohammed-Abrar-Neeman/crypto-convertor-ui.git)

UI live server
[https://crypto-convertor-ui.onrender.com/](https://crypto-convertor-ui.onrender.com/)

## Hosted to Render

To get Coins list[https://crypto-convertor.onrender.com/v1/priceConverter/coinsList](https://crypto-convertor.onrender.com/v1/priceConverter/coinsList)

To get fiat list [https://crypto-convertor.onrender.com/v1/priceConverter/currencyList](https://crypto-convertor.onrender.com/v1/priceConverter/currencyList)

To get price from crypto to fiat 

[https://crypto-convertor.onrender.com/v1/priceConverter/getPrice?coin=ethereum&fiat=inr&amount=1](https://crypto-convertor.onrender.com/v1/priceConverter/getPrice?coin=ethereum&fiat=inr&amount=1)


## Running Development Server

```bash
yarn

yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
